# ecommerce-syntaxwear

Projeto inicial para um e-commerce fictício chamado **SyntaxWear**.

Descrição:
- Estrutura base com pastas para assets (CSS, imagens e fontes), páginas e documentação.
- Arquivos CSS organizados por camadas: `00-settings`, `01-base`, `02-layout`, `03-components`, `04-utilities`.
- Página principal `index.html` com template inicial e links para os estilos.

Objetivo:
Fornecer um esqueleto inicial para desenvolvimento de um frontend de loja online, facilitando organização de estilos e assets.

Como usar:
1. Abrir o projeto no editor.
2. Editar `index.html` e os arquivos dentro de `assets/css/` para começar a implementar o layout.
3. Colocar imagens em `assets/img/` e fontes em `assets/fonts/`.
